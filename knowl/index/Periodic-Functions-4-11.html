Activity 4.5. Unit Circles.

  1. Use a protractor to draw an angle \(36\degree\) in standard position.
    1. Estimate the coordinates of the point \(P\) where the terminal side of the angle intersects the circle of radius \(r=2\text{.}\)
    2. Calculate approximate values for \(\cos (\theta)\) and \(\sin (\theta)\) using the coordinates of \(P\text{.}\)
    3. Estimate the coordinates of the point \(Q\) where the terminal side of the angle intersects the circle of radius \(r=1\text{.}\)
    4. Calculate approximate values for \(\cos (\theta)\) and \(\sin (\theta)\) using the coordinates of \(Q\text{.}\)
  2. Use a protractor to draw an angle \(107\degree\) in standard position. Repeat parts (a)-(d) for this new angle.
  3. Use a protractor to draw an angle \(212\degree\) in standard position. Repeat parts (a)-(d) for this new angle.
  4. Use a protractor to draw an angle \(325\degree\) in standard position. Repeat parts (a)-(d) for this new angle.
  5. What do you notice about the coordinates of the point located on the unit circle by an angle and the values of the trig ratios of that angle?
