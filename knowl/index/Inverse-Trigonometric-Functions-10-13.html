Definition 8.35. The inverse tangent function.

The function \(f(x)=\tan^{-1}(x)\) is defined as follows:
\begin{equation*} \blert{\tan^{-1}(x)=\theta}~~~\text{if and only if}~~~~\blert{\tan (\theta) = x}~~~\text{and}~~~\blert{\dfrac{-\pi}{2} \lt \theta \lt \dfrac{\pi}{2}} \end{equation*}
