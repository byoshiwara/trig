Apply Property (2) to write the radical as a quotient. \(\begin{aligned} \sqrt{\frac{1}{3}}\amp = \frac{\sqrt{1}}{\sqrt{3}} \\ \amp =\frac{1}{\sqrt{3}} \amp\amp\text{Multiply numerator and denominator by } \sqrt{3}.\\ \amp = \frac{1\cdot \alert{\sqrt{3}}}{\sqrt{3}\cdot \alert{\sqrt{3}}}\\ \amp = \frac{\sqrt{3}}{3} \end{aligned}\)
