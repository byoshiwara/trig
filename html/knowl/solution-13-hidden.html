We substitute \((2,-1)\) for \((x_1, y_1)\) and \((4,3)\) for \((x_2, y_2)\) in the distance formula to obtain

\begin{align*} d\amp = \sqrt{(x_2-x_1)^2+(y_2-y_1)^2}\\ \amp = \sqrt{(4-2)^2+[3-(-1)]^2}\\ \amp = \sqrt{4+16} = \sqrt{20} = 2\sqrt{5}\\ \end{align*} The exact value of the distance, shown at right, is \(2\sqrt{5}\) units. We obtain the same answer if we use \((4,3)\) for \(P_1\) and \((2,-1)\) for \(P_2\text{:}\) \begin{align*} d\amp = \sqrt{(2-4)^2+[(-1)-3]^2}\\ \amp = \sqrt{4+16} = \sqrt{20} = 2\sqrt{5}\\ \end{align*}

We can use a calculator to obtain an approximation for this value, and find

\begin{equation*} 2\sqrt{5}\approx 2(2.236)=4.472 \end{equation*}
